Guides extension

1. Guides

  • B1. Bạn tạo một Spreadsheet mới, đặt tên tùy ý, ở đây mình đặt là English

  • B2. Click vào nút copy ở bên dưới,quay lại trang Spreadsheet mới tạo ,đặt con trỏ chuột ở A1, rồi bấm Ctrl + V

  • B3. Đổi sheet name là vocabulary

  • B4. Trên thanh bar bạn click lần lượt Tools >> Script Editor

  • B5. Xóa 3 dòng đầu đi

  • B6. CLick vào nút copy bên dưới, vào lại chỗ bạn mới vừa xóa, rồi bấm Ctrl + V, sau đó bấm Ctrl + S để lưu lại

  • B7. Ở bên phải màn hình vào Deploy >> New Deployment

  • B8. Bạn click vào hình răng cưa, click web app

  • B9. Bạn chọn các thông tin như dưới ảnh, phần description gõ app rồi click deploy

  • B10. Bạn lick vào Authorize access >> Chọn tài khoản của bạn >> Nâng cao >> Đi tới dự án... hoặc Authorize access >> Chọn tài khoản của bạn >> cho phép

  • B11. Click copy như hình phía dưới

  • B12. Mở extension lên, chuyển qua tab API, Ctrl + V vào ô, bấm ADD API
